# Hospital Management System (Java)

## Project Overview
This is a console-based Hospital Management System developed using Java and Object-Oriented Programming concepts.

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Interface
- Packages
- Exception Handling

## Modules Description
- Person: Base class
- Doctor & Patient: Derived classes
- Manageable: Interface
- HospitalService: Business logic
- HospitalApp: Main menu-driven application

## How to Run
1. Compile the program using javac
2. Run HospitalApp.java
3. Follow on-screen menu
